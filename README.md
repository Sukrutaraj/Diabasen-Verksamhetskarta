# Diabasen-Verksamhetskarta
En Digital Visningsmiljö för Diabasen – en interaktiv verksamhetskarta som gör det möjligt att utforska vår verksamhet digitalt.  Här synliggörs våra lokaler, vår kompetens, vår teknik och hur vi arbetar – på ett sätt som gör det enkelt att förstå helheten, både för nya deltagare, ny personal och vid studiebesök.
